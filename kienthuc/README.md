# Khu vực Kiến thức

## Cấu trúc

- `index.html`: trang thư viện và tìm kiếm.
- `articles/devops/`: Linux, Docker, Cloud, Network, Terraform, Nginx và CI/CD.
- `articles/aws/`: các bài AWS và dịch vụ liên quan.
- `articles/ai/`: AI, Machine Learning và Deep Learning.
- `articles/programming/`: máy tính và ngôn ngữ lập trình.
- `articles/web/`: nền tảng và hiệu năng web.
- `assets/css/`: giao diện dùng chung.
- `assets/js/knowledge-data.js`: danh mục trung tâm của toàn bộ bài viết.
- `assets/decor/`: ảnh trang trí dùng trong khu vực Kiến thức.
- `0/` và `1/`: chỉ giữ các trang chuyển hướng để URL cũ tiếp tục hoạt động.

Khi thêm bài mới, tạo file trong nhóm tương ứng rồi thêm metadata vào `assets/js/knowledge-data.js` và thẻ bài ở `index.html`.
